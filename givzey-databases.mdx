### **S3 Data Storage**

The diagram also shows various S3 buckets organized into categories:

1. **Database Snapshots & Exports**:  
   * givzey-db-snapshot: For database exports and backups  
   * version2-dev-db: For Version2 database files  
2. **Data Transfer**:  
   * givzey-sftp-record-transfer: For client data transfers  
   * version2-ai: For customer data transfers  
   * version2-ai-dev: For development data  
3. **Application Assets**:  
   * givzey-assets: For static assets  
   * givzey-dev: For development assets  
4. **Knowledge Bases**:  
   * version2-kb-organization  
   * version2-ses-email-forwarding
